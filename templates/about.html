JSMusicDB is a webstandards based music library/music player to be used in conjuction to the scanner.py script that can run on a NAS, like any synology NAS.

Any type of scanning script can be used as long as the output is a JSON file with a specific layout.

JSMusicDB is in public beta. Any bugs found can be reported but some features are already in scope.

The following features are currently in scope:

  • Ability to create new playlists
  • Import/export playlists
